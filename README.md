# Event by Jaleco

A boutique wedding planning landing page built with Next.js 14, TypeScript, and Tailwind CSS.

## Design

- **Style**: Soft UI / Neumorphism-lite with editorial typography
- **Palette**: Warm pastel white with champagne gold and blush rose accents
- **Typography**: Cormorant Garamond (display) · Great Vibes (script) · Raleway (body)

## Sections

- Hero with dual CTAs
- About / story
- Services (3 packages)
- Portfolio gallery
- Testimonials
- FAQ accordion
- Contact form
- Footer

## Local Development

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run start
```
