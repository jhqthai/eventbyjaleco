## Design Philosophy by Vertical

### Trust-Critical Verticals (Legal, Finance, Healthcare, Government)
- Always use high contrast (WCAG AAA preferred)
- Limit decorative effects
- Prioritize accessibility
- Navy/Blue primaries convey trust
- Gold accents for premium tier

### Engagement-Critical Verticals (Social, Gaming, Entertainment)
- Maximize visual impact and color
- Motion and micro-interactions essential
- Dark modes standard for media consumption
- Gamification elements (streaks, badges, progress)

### Conversion-Critical Verticals (E-commerce, SaaS, Services)
- High-contrast CTAs above fold
- Social proof visible before CTA
- Progress/trust indicators in checkout
- Mobile-first layout

### Content-Critical Verticals (Media, News, Education)
- Typography-first design
- Reading mode / focus mode
- Fast loading (minimal effects)
- Dark/light toggle
