# Typography Pairings — 73 Google Fonts

Format: `No | Name | Category | Heading Font | Body Font | Best For | Google Fonts CSS Import`

All fonts available on fonts.google.com. Tailwind config format: `fontFamily: { heading: ['Font Name', 'sans-serif'], body: ['Font Name', 'sans-serif'] }`

> **⚠ Dated for 2026 — do not pair these on luxury, editorial, wedding, or premium hospitality briefs:**
> - **Script display fonts**: Great Vibes, Allura, Pacifico, Lobster, Sacramento, Dancing Script — they read as 2010s Etsy/wedding-template. Replace with *Cormorant Garamond Italic* or *Editorial New Italic* at the same display size.
> - **Raleway** as the body sans on premium brands — its 2014 web-default association feels dated. Replace with *Inter*, *Söhne*, or *GT America*.
> - **Comic-rounded** sans (Quicksand, Comfortaa, Nunito) on anything serious — reserve for kids / education only.
> - For modern luxury wedding/event sites, the canonical pairing is **Cormorant Garamond (with italic accent) + Inter**. See `references/quiet-luxury-2026.md`.

## CSS Import Templates

### Standard Dual-Font (most common):
```css
@import url('https://fonts.googleapis.com/css2?family=HEADING:wght@400;600;700&family=BODY:wght@400;500&display=swap');
```

### For Languages with Extended Character Support:
```css
/* Vietnamese */
@import url('https://fonts.googleapis.com/css2?family=Be+Vietnam+Pro:wght@300;400;500;600;700&family=Noto+Sans:wght@300;400;500;600;700&display=swap');
/* Japanese */
@import url('https://fonts.googleapis.com/css2?family=Noto+Serif+JP:wght@400;500;600;700&family=Noto+Sans+JP:wght@300;400;500;700&display=swap');
/* Korean */
@import url('https://fonts.googleapis.com/css2?family=Noto+Sans+KR:wght@300;400;500;700&display=swap');
/* Arabic (RTL) */
@import url('https://fonts.googleapis.com/css2?family=Noto+Naskh+Arabic:wght@400;500;600;700&family=Noto+Sans+Arabic:wght@300;400;500;700&display=swap');
```

### RTL CSS Setup (Arabic/Hebrew):
```css
html[dir="rtl"] {
  direction: rtl;
  text-align: right;
}
```
