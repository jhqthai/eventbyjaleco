## Detailed Patterns

### 1. Hero + Features + CTA (Standard)
**Section order**: Hero with headline/image → Value prop → Key features (3–5) → CTA section → Footer  
**CTA placement**: Hero (sticky) + Bottom  
**Color strategy**: Hero: Brand primary. Features: Card bg #FAFAFA. CTA: Contrasting accent (7:1+ contrast ratio)  
**Effects**: Hero parallax, feature card hover lift, CTA glow on hover  
**Conversion tips**: Deep CTA placement. Use contrasting color. Sticky navbar CTA.

### 2. Hero + Testimonials + CTA (Social Proof)
**Section order**: Hero → Problem statement → Solution → Testimonials carousel → CTA  
**CTA placement**: Hero (sticky) + Post-testimonials  
**Color strategy**: Hero: Brand. Testimonials: #F5F5F5. Quotes: italic muted #666. CTA: Vibrant  
**Effects**: Testimonial carousel slide, quote marks animation, avatar fade-in  
**Conversion tips**: Use 3–5 testimonials. Include photo + name + role. CTA after social proof.

### 3. Product Demo + Features
**Section order**: Hero → Product video/mockup (center) → Feature breakdown → Comparison (optional) → CTA  
**CTA placement**: Video center + CTA right/bottom  
**Color strategy**: Video surround: Brand overlay. Features: Icon color #0080FF. Text: Dark #222  
**Effects**: Video play button pulse, feature scroll reveals, demo interaction highlights  
**Conversion tips**: Auto-play video muted. Use interactive mockup. Embedded demo increases engagement.

### 4. Minimal Single Column
**Section order**: Hero headline → Short description → Benefit bullets (max 3) → CTA → Footer  
**CTA placement**: Center, large CTA button  
**Color strategy**: Brand + white #FFFFFF + accent. Buttons: 7:1+ contrast. Text: Black/Dark grey  
**Effects**: Minimal hover. Smooth scroll. CTA scale on hover (subtle)  
**Conversion tips**: Single CTA focus. Large typography. Lots of whitespace. Mobile-first.

### 5. Funnel (3-Step Conversion)
**Section order**: Hero → Step 1 (problem) → Step 2 (solution) → Step 3 (action) → CTA progression  
**CTA placement**: Each step: mini-CTA. Final: main CTA  
**Color strategy**: Step 1 Red/Problem → Step 2 Orange/Process → Step 3 Green/Solution. CTA: Brand  
**Effects**: Step number animations, progress bar fill, smooth scroll transitions  
**Conversion tips**: Progressive disclosure. Show only essential info per step. Use progress indicators.

### 6. Comparison Table + CTA
**Section order**: Hero → Problem → Comparison table (product vs competitors) → Pricing (optional) → CTA  
**CTA placement**: Table right column + Below table  
**Color strategy**: Table: Alternating rows white/light grey. Your product: Highlight #FFFACD or green. CTA: Vibrant  
**Effects**: Table row hover highlight, price toggle animation, feature checkmark animation  
**Conversion tips**: Highlight your product row. Include 'free trial'. Be factual. +35% higher conversion.

### 7. Lead Magnet + Form
**Section order**: Hero (benefit headline) → Lead magnet preview → Form (≤3 fields) → CTA submit  
**CTA placement**: Form submit button  
**Color strategy**: Lead magnet professional. Form: clean white. Inputs: #CCCCCC border. CTA: Brand  
**Effects**: Form focus state animations, input validation, success confirmation animation  
**Conversion tips**: Form fields ≤3 for best conversion. Show lead magnet preview. Progress feedback.

### 8. Pricing Page + CTA
**Section order**: Hero (pricing headline) → Price comparison cards → Feature table → FAQ → Final CTA  
**CTA placement**: Each card CTA + Sticky nav + Bottom  
**Color strategy**: Free: Grey. Starter: Blue. Pro: Green/Gold. Enterprise: Dark. Cards: 1px border, shadow  
**Effects**: Price toggle animation (monthly/yearly), card hover lift, FAQ accordion  
**Conversion tips**: Pre-select recommended plan. Annual discount 20–30%. FAQs address objections.

### 9. Video-First Hero
**Section order**: Hero with video background → Key features overlay → Benefits → CTA  
**CTA placement**: Overlay on video (center/bottom) + Bottom section  
**Color strategy**: Dark overlay 60% on video. Brand accent CTA. White text on dark  
**Effects**: Video autoplay muted, parallax scroll, text fade-in on scroll  
**Conversion tips**: 86% higher engagement. Add captions for accessibility. Compress video.

### 10. Scroll-Triggered Storytelling
**Section order**: Intro hook → Chapter 1 (problem) → Chapter 2 (journey) → Chapter 3 (solution) → Climax CTA  
**CTA placement**: End of each chapter (mini) + Final climax  
**Color strategy**: Progressive reveal. Each chapter distinct color. Building intensity  
**Effects**: ScrollTrigger animations, parallax layers, progressive disclosure, chapter transitions  
**Conversion tips**: Narrative increases time-on-page 3×. Use progress indicator. Mobile: simplify animations.

### 11. Waitlist / Coming Soon
**Section order**: Hero with countdown → Product teaser → Email capture form → Social proof (waitlist count)  
**CTA placement**: Email form prominent (above fold) + Sticky on scroll  
**Color strategy**: Anticipation: Dark + accent highlights. Countdown in brand color. Urgency indicators  
**Effects**: Countdown timer, email validation feedback, success confetti, social share buttons  
**Conversion tips**: Scarcity + exclusivity. Show waitlist count. Early access benefits. Referral program.

### 12. Pricing-Focused Landing
**Section order**: Hero (value prop) → Pricing cards (3 tiers) → Feature comparison → FAQ → Final CTA  
**CTA placement**: Each pricing card + Sticky nav + Bottom  
**Color strategy**: Popular plan: brand color border/bg. Free: grey. Enterprise: dark/premium  
**Effects**: Price toggle monthly/annual, card hover lift, FAQ accordion  
**Conversion tips**: Annual discount 20–30%. Recommend mid-tier (most popular badge). Address objections.

### 13. App Store Style
**Section order**: Hero with device mockup → Screenshots carousel → Features with icons → Reviews/ratings → Download CTAs  
**CTA placement**: Download buttons prominent throughout (App Store + Play Store)  
**Color strategy**: Dark/light matching app store feel. Star ratings in gold. Device frames  
**Effects**: Device mockup rotations, screenshot slider, star rating animations, download button pulse  
**Conversion tips**: Show real screenshots. Include ratings (4.5+ stars). QR code for mobile.

### 14. FAQ/Documentation Landing
**Section order**: Hero with search bar → Popular categories → FAQ accordion → Contact/support CTA  
**CTA placement**: Search bar prominent + Contact CTA for unresolved questions  
**Color strategy**: Clean, high readability. Minimal color. Category icons in brand. Success green for resolved  
**Effects**: Search autocomplete, smooth accordion, category hover, feedback buttons  
**Conversion tips**: Reduce support tickets. Track search analytics. Show related articles.

### 15. Event/Conference Landing
**Section order**: Hero (date/location/countdown) → Speakers grid → Agenda/schedule → Sponsors → Register CTA  
**CTA placement**: Register CTA sticky + After speakers + Bottom  
**Color strategy**: Urgency colors (countdown). Event branding. Speaker cards professional. Sponsor logos neutral  
**Effects**: Countdown timer, speaker hover cards with bio, agenda tabs, early bird countdown  
**Conversion tips**: Early bird pricing with deadline. Social proof (past attendees). Multi-ticket discounts.

### 16. Interactive Product Demo
**Section order**: Full-screen interactive element → Guided tour → Key benefits revealed → CTA after completion  
**CTA placement**: After interaction complete + Skip option  
**Color strategy**: Immersive experience. Dark background for focus. Highlight interactive elements  
**Effects**: WebGL, 3D interactions, gamification, progress indicators, reward animations  
**Conversion tips**: 40% higher engagement. Provide skip option. Mobile fallback essential.

### 17. Community / Forum Landing
**Section order**: Hero (community value prop) → Popular topics → Active members → Join CTA  
**CTA placement**: Join button persistent + After each section  
**Conversion tips**: Show member count + active discussions. Highlight exclusive content benefits.

### 18. Trust & Authority (Professional Services)
**Section order**: Hero (credentials) → Social proof (logos/awards) → Services → Team → Case studies → CTA  
**Color strategy**: Professional blue/navy + neutral. Trust badges prominent. High contrast text  
**Conversion tips**: Credibility signals above fold. Clear contact/CTA. Phone number visible.

### 19. Conversion-Optimized (Direct Response)
**Section order**: Headline + subheadline → Urgency element → CTA → Benefits → FAQ → Final CTA  
**CTA placement**: Multiple — top, middle, bottom + sticky  
**Color strategy**: High contrast CTA. Urgency in red/orange. Benefits in green checkmarks  
**Effects**: Countdown timer, limited spots indicator, social proof ticker  
**Conversion tips**: Remove navigation. Reduce distractions. A/B test headline. 1 goal only.

### 20. Storytelling-Driven (Narrative)
**Section order**: Emotional hook → Problem narrative → Journey → Transformation → CTA  
**Color strategy**: Emotional progression. Warm for problem, cool for resolution, vibrant for CTA  
**Conversion tips**: Protagonist = the user. Show before/after. Use real stories with photos.

## CTA Design Rules

1. **Single primary CTA** per page section — remove competing buttons
2. **Minimum button size**: 44×44px touch target (mobile)
3. **Contrast ratio**: 4.5:1 minimum; 7:1 ideal for primary CTA
4. **Action words**: "Get Started", "Try Free", "Download", "Book Now" — not "Submit"
5. **Sticky CTAs**: Use on long pages — after user scrolls 60% of content
6. **Above the fold**: Primary CTA visible without scrolling on all devices

## Performance + Accessibility

- Use `loading="lazy"` on all images below fold
- Hero images: provide `alt` text and `width`/`height` attributes
- Video: always include `muted`, `playsinline`, and `<track>` for captions
- Animations: wrap in `@media (prefers-reduced-motion: reduce)` check
- Form fields: always associate `<label>` elements
